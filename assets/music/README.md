# assets/music

Drop **rights-cleared** audio tracks here (`.mp3`, `.m4a`, `.wav`, `.aac`).

`video_builder.py` will randomly pick one per video and loop/fade it under the
visuals. If this folder is empty, videos are rendered with a **silent** audio
track (still valid MP4, still uploadable).

⚠️ **The system never downloads music automatically.** Only use audio you are
licensed to use commercially on TikTok/YouTube — e.g.:

- Tracks from the platforms' own Commercial Music Libraries (added in-app after upload)
- Creative Commons / royalty-free libraries you have cleared
- Music you own or have a licence for

Set a specific default track via `config.yaml → video.default_music`.
